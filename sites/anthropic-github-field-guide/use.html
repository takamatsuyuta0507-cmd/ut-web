つかうUse in your work — 11 repositories

コードを書かなくても使える領域。Skills・Plugins・Coworkで日々の業務を定型化・自動化し、業界特化ソリューション（法務・金融・医療など）で専門業務に導入します。

Skills・Plugins・Cowork

· リポジトリ

skills は全リポジトリ最多スター。Claudeに自社の手順を覚えさせる「スキル」、機能を足す「プラグイン」、業務に常駐させる「Cowork」。非エンジニアの入口にも最適です。

業界特化

· リポジトリ

業界の業務に合わせて作られたプラグイン／マーケットプレイス。自社の領域に近いものから導入を検討。

💼 実務メモ — 非エンジニアの始め方

  • 1. cwc-workshops でCoworkの基本をハンズオンで体験。
  • 2. knowledge-work-plugins から業務に合うプラグインを導入。
  • 3. claude-plugins-official の厳選プラグインで機能を拡張。
  • 4. 繰り返す手順は skills で「スキル」化して定型化。

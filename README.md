# ut-web

UT.（東京の一人スタジオ）の **公開ウェブ** をまとめた場所です。
ポートフォリオ、会社サイト、そして単機能アプリ群 **koto** ── 外に出している制作物だけを置いています。

哲学は **「少なく、よく」**。
ビルドなし。フレームワークなし。手で書いた HTML / CSS / JS だけ。

## ビルドしません

## ローカルで見る

どれかのフォルダで、`index.html` をブラウザで開くだけ。
パスを正しく見たいときは、小さな静的サーバーを **5070番台** のポートで：

```bash
# 例：あけてね を 5074 番で
python -m http.server 5074 --directory apps/koto-akete
# → http://localhost:5074
```

> ポート 5060 / 5061 はブラウザ側でブロックされます。5070番台を使ってください。

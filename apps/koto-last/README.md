# Last

最後にやったの、いつ？ を記録する。

やったらタップ。経過日数が大きく出る。それだけ。
目安の間隔を決めておくと、超えたぶんがオレンジになる。

koto シリーズの1つ（UT. の単機能アプリ群）。**公開準備中。**

---

## ひとつのことだけ

「最後にやったの、いつ？」を覚えておく。
換気扇そうじ、コンタクト交換、洗車、観葉植物の水やり——
忘れがちで、頻度が決まっていない用事のための小さな道具。

## つくり

- ビルドなし。素の HTML / CSS / JS。
- PWA。一度開けばオフラインでも動く（Service Worker）。
- 記録はこの端末の中だけ（localStorage）。**サーバーには何も送らない。**

## はじめ方

静的サーバーで開くだけ。ポートは 5070 番台で（5060/5061 は Chrome がブロックする）。

```sh
python -m http.server 5070
# → http://localhost:5070
```

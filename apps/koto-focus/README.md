# Focus — シリーズ第1号アプリ

1つのことに集中するための、シンプルなタイマー。
白を基調に、差し色のオレンジを1点だけ。ビルド不要の静的サイト（PWA対応）。

## ブランド名の差し替え
`app.js` の先頭、**1行だけ**変えればロゴ表記が切り替わります。
```js
const BRAND = "koto.";   // 例: "dot." / "mono." など。ピリオドが自動でオレンジに。
```

## ローカルで開く
ビルド不要。簡易サーバーで配信するだけ（Service Worker のため file:// ではなく http:// 推奨）。
```bash
npx serve .        # もしくは python -m http.server
```

## デプロイ（Cloudflare Pages）
このフォルダをそのまま Pages にアップロード／接続するだけ。ビルドコマンドは不要、出力ディレクトリはこのフォルダ。

## ファイル構成
- `index.html` — マークアップ
- `styles.css` — デザインシステム（カラー変数は `:root`）
- `app.js` — タイマーのロジック（`BRAND` 定数もここ）
- `manifest.webmanifest` / `sw.js` — PWA（インストール・オフライン）
- `icon.svg` — アプリアイコン（シリーズ共通テンプレ：白角丸＋中央のオレンジ）

## iOS への布石
- PWA としてホーム画面に追加可能。
- 将来 SwiftUI に移すとき、`:root` のカラー変数と `icon.svg` のテンプレをそのまま流用できます。
- iOS のホーム画面アイコンを綺麗に出すには PNG（180×180 等）が必要 → 「ロゴ・アイコン体系」ステップでまとめて書き出します。
